Matri-seq Example¶

Import the RNAscrutiny package and initialize with init function. Here, we set the output directory to a new directory with the current timestamp, and we set the verbosity of the package to False.

In [1]: import RNAscrutiny as scrutiny

In [2]: import datetime

In [3]: scrutiny.init(outputDir=datetime.datetime.now().strftime('%d_%m_%Y-%H:%M:%S'), verbose=False)

Use the generateCorrlationMatrix function to generate a gene correlation matrix. The networkStructure can be one of SimulatedPowerLaw, TRRUST, or Random. n denotes the number of genes.

In [4]: networkStructure = 'SimulatedPowerLaw'

In [5]: n = 700

In [6]: gc, tf = scrutiny.generateGeneCorrelationMatrix(n=n, networkStructure=networkStructure)

Represent the cell development as a tree with the formCellTypeTree function.

In [7]: cells = 200

In [8]: cellTypeTree, projMatrix, constMatrix = scrutiny.formCellTypeTree(n=n, cells=cells, cellTypeParents=[-1], cellTypeNumCells=[200], cellTypeConstProps=[0.0], transcriptionFactors=tf, cellTypeMeans=[0])

Determine the minmum value of alpha that allows convergence of the cell states.

In [9]: initialStates = scrutiny.generateInitialStates(n, cells)

In [10]: alpha = scrutiny.findOptimalAlpha(n, gc, cellTypeTree, initialStates, numToSample=50)

In [11]: gc /= alpha

Recurse over cell type tree to develop cells.

In [12]: currentStates = initialStates

In [13]: cellTypesRecord = np.zeros(shape=cells, dtype="int64")

In [14]: totalCellIterations = np.zeros(shape=cells, dtype="int64")

In [15]: scrutiny.findAllNextStates(gc, cellTypeTree.head(), cellTypeTree, currentStates, cellTypesRecord, totalCellIterations)

In [16]: finalCellStates = currentStates

Output plots to the output directory set above.

In [17]: cellSizeFactors = np.zeros(shape=cells)

In [18]: timeStep = 0.01

In [19]: pseudotimes = timeStep * totalCellIterations

In [20]: scrutiny.analyzeDataBeforeTransformation(n, cells, finalCellStates, alpha, pseudotimes, networkStructure=networkStructure)

Get the final cell states

In [21]: finalCellStates, cellSizeFactors = scrutiny.transformData(n, cells, finalCellStates)

Output plots to the output directory set above:

In [22]: scrutiny.analyzeSCRNAseqData(n, cells, cellTypesRecord, finalCellStates)
